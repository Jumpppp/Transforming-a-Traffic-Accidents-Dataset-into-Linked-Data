# Traffic Accidents Linked Data Project

This repository contains the triplified version of a traffic accidents dataset.

## Files
- `rdf/accidents.ttl` — RDF triples in Turtle format
- `data/raw/accidents.csv` — source dataset
- `report/report.pdf` — report on the triplification process

## Vocabularies used
- schema.org
- RDF
- XSD
- SKOS
- custom namespace: `ex:`
